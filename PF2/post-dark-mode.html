Dark mode improves readability in low light. In this tutorial, we'll implement it with CSS variables and JavaScript.

1. Define CSS variables

:root {
  --bg: #f9f9f9; --fg: #222;
}
[data-theme="dark"] {
  --bg: #121212; --fg: #eee;
}
body { background: var(--bg); color: var(--fg); transition: background .4s, color .4s; }
